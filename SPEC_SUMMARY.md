# SPEC SUMMARY - SOS Web

SOS Web è la nuova versione web + database SQL di SOS Recuperi.

## Ruoli previsti

- ADMIN
- DOCENTE
- STUDENTE
- NON_AUTORIZZATO

## Entità principali

- utenti
- docenti
- studenti
- materie
- attività/pacchetti recupero
- lezioni
- iscrizioni
- notifiche/log
- configurazione

## Prima fase

La prima fase crea ambiente portabile, backend health check, connessione DB, schema iniziale, frontend base e test smoke Playwright.
