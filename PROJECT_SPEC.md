# PROJECT SPEC - SOS Web

## Scopo

Realizzare una web app portabile con server applicativo e database SQL per gestire SOS, sportelli didattici e recuperi estivi.

## Prima fase

- Setup ambiente sviluppo.
- Backend API minimale.
- Database MariaDB/MySQL.
- Schema Prisma iniziale.
- Frontend Vue base.
- Test E2E smoke.
- Documentazione operativa.

## Non obiettivi iniziali

- Migrazione dati completa.
- Deploy produzione.
- Integrazione OAuth completa.
- Notifiche email reali.
- Dashboard avanzate.
